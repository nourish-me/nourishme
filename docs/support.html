Häufige Fragen

Werden meine Daten an Server gesendet?

Deine Eingaben (Mahlzeit-Text, optionale Fotos) werden anonym an die Anthropic Claude AI gesendet, um die Coaching-Antwort zu erzeugen. Sonst nichts. Details in der Datenschutzerklärung.

Kann ich meine Daten exportieren?

Aktuell gibt es keine Export-Funktion in der App. Auf Anfrage per E-Mail können wir dir deine lokalen Daten zugänglich machen.

Wie lösche ich alle meine Daten?

In den Einstellungen ganz unten: „App zurücksetzen". Alternativ: App deinstallieren.

Ist die App medizinisch geprüft?

Nein. NourishMe ist ein persönliches Wellness-Tool, das auf wissenschaftlichen Empfehlungen (DGE, BfR, EFSA, LactMed, FDA/EPA) basiert. Es ersetzt keine ärztliche Beratung. Bei medizinischen Fragen wende dich an deine Ärztin oder Hebamme.

Warum gibt der Coach manchmal leicht unterschiedliche Werte für die gleiche Zutat?

Die Nährwerte werden von einer KI geschätzt, nicht aus einer Datenbank. Je nach Kontext und Formulierung können die Schätzungen leicht variieren. Eine Integration einer Lebensmittel-Datenbank ist geplant.
