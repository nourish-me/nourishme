# Screenshots needed for the Claude Design handoff

Drop iOS screenshots here (PNG, full-screen, from your iPhone 13 or
the iPhone 16e simulator at 393pt width). The Design Prompt v2
references them by the filenames below — keep these names so the
prompt matches.

## Required

1. **`diary-today.png`** — Diary tab, today's view, showing the
   AppBar with the existing kcal + macros toolbar AND at least one
   logged meal. This defines what's already taking up vertical
   space at the top of the screen, so the new micronutrient strip
   doesn't collide.

2. **`diary-with-coach.png`** — Diary view with one meal + its
   coach reply bubble visible. Establishes the existing visual
   language (amber container, in-thread bubble) the strip needs to
   coexist with.

3. **`settings-main.png`** — The main Settings screen so the new
   "Daily supplement" sub-section uses the same card/list pattern.

4. **`onboarding-phase-details.png`** — The Phase-Details step of
   onboarding (currently step 4 of 5) so the new Supplement step
   matches the same stepper/Card aesthetic.

## Optional but helpful

5. **`tips-deck.png`** — One of the Tips-Screen cards, to see how
   illustrations + body text get treated when they're a "card" the
   user dwells on (for the future detail-modal layout).

## How to take them

iOS Simulator: open the app, navigate, then File → Save Screen
(Cmd+S) — saves a full-resolution PNG to your Desktop. Drag here.

Physical iPhone: side button + volume up = screenshot, then AirDrop
to your Mac and drag here.

## Then

Once all 4 required PNGs are here, upload them as attachments to
your Claude Design conversation along with `../design-prompt-v2.md`.
The prompt explicitly tells Claude Design they exist and how to
read them.
